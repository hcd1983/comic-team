# 素材搜集

在編劇討論前搜集外部素材，為編劇提供創作燃料。

## 流程

1. 詢問使用者（若未提供）：
   - 故事主題或關鍵字
   - 目標受眾（可選，預設「通用」）
   - 漫畫調性（搞笑 / 熱血 / 溫馨 / 懸疑）
2. 呼叫 `researcher` agent（小查），傳入主題、受眾、調性
3. 小查透過 WebSearch 搜集四大類素材：
   - 熱門梗/迷因
   - 經典橋段
   - 社群討論
   - 受眾吐槽點
4. 呈現素材包摘要給使用者：

```
═══ 素材搜集完成 ═══

📊 搜集概覽：
  熱門梗/迷因：N 條
  經典橋段：N 條
  社群討論：N 條
  受眾吐槽點：N 條

🔑 最值得利用的發現：
  1. ...
  2. ...
  3. ...

⚠️ 避免清單（已被用爛的梗）：
  1. ...

💡 建議創作切角：
  1. ...
  2. ...

───────────────────────────────────
請選擇：確認 / 追加搜尋特定方向 / 重新搜尋 / 跳過
```

5. 使用者選擇：
   - **確認** → 儲存至 `output/material-pack.json`
   - **追加搜尋特定方向** → 使用者指定方向，小查追加搜尋並合併結果
   - **重新搜尋** → 重新呼叫小查
   - **跳過** → 不使用素材包，直接進入編劇討論

6. 確認後告知使用者檔案路徑。
